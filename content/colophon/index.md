A short note on how this site is built, in the spirit of the colophon tradition.

## Stack

- **[Quartz](https://quartz.jzhao.xyz)** — the static site generator. Every page is a Markdown file; the site is a Quartz vault.
- **[Pandoc](https://pandoc.org)** — underlying converter, handling citations and cross-references.
- **[SCSS + Bootstrap](https://getbootstrap.com)** — theme layer. Light theme inherits Quartz's default; dark theme extends *darkly*. The palette (deep navy, soft sage, warm taupe, cream) lives in `custom.scss` in the repository root.
- **Type:** body, headings, and italic emphasis in **[Atkinson Hyperlegible Next](https://fonts.google.com/specimen/Atkinson+Hyperlegible+Next)** by the [Braille Institute](https://www.brailleinstitute.org/freefont/), low-vision-tuned and variable on a single weight axis (200–800). Code: **[JetBrains Mono](https://www.jetbrains.com/lp/mono/)**, also variable. Both families are self-hosted as woff2 to avoid a third-party font CDN.

## Source and hosting

- **Source:** a [GitHub repository](https://github.com/tiagojct/tiagojct-site). Each `git push` to `main` triggers a build.
- **CI/CD:** [GitHub Actions](https://github.com/tiagojct/tiagojct-site/blob/main/.github/workflows/deploy.yml) builds the site and deploys it.
- **Hosting:** [GitHub Pages](https://docs.github.com/en/pages) serves the static output directly.
- **Domain:** [`tiagojct.eu`](https://tiagojct.eu), with a CNAME record pointing to GitHub Pages.

## Feeds and metadata

- **RSS feeds** for [Notes](/notes/index.xml), [Essays](/essays/index.xml), and [Keynotes](/keynotes/index.xml), generated by Quartz from the listing pages.
- **Sitemap** at [`/sitemap.xml`](/sitemap.xml), auto-generated.
- **Open Graph** and **Twitter Card** metadata on every page, so shared links render rich previews.

## Writing and review

All pages are written in plain-text Markdown, edited in [Zettlr](/notes/zettlr/) or [Visual Studio Code](https://code.visualstudio.com/), and spell-checked by eye. Some of the longer notes were drafted with help from [Claude](https://claude.ai) (via [Claude Code](https://www.anthropic.com/claude-code) and [OpenCode](https://opencode.ai/)); every line is reviewed, edited, and signed off by me. The [scientific writing note](/notes/sw/) sets out exactly how AI assistance is disclosed in my own writing.

## Analytics

Page views are counted with [Google Analytics 4](https://marketingplatform.google.com/about/analytics/). Aggregated traffic only. If you would prefer your visit not be measured, a tracker-blocking extension or browser setting will stop the request from firing.
